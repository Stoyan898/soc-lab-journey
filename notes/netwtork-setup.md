# Network Configuration

## Objective
Ensure communication between Windows and Ubuntu virtual machines.

## Steps
- Checked IP address on Ubuntu using:
  ifconfig

- Checked IP on Windows using:
  ipconfig

- Tested connectivity:
  ping 192.168.1.3

## Result
- Successful communication between both machines
- 0% packet loss

## Key Learning
Understanding how devices communicate over a network is essential for SOC roles, especially for detecting abnormal traffic.
