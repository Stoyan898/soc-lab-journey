# SOC Lab Journey

## 📌 Overview

This repository documents my hands-on journey building a Security Operations Center (SOC) lab using Splunk.

## 🛠️ Tools Used

* Splunk Enterprise (Ubuntu)
* Splunk Universal Forwarder (Windows)
* Windows Event Logs

## 🔍 Detections Implemented

* Brute Force Detection
* Suspicious Login After Failed Attempts

## 🎯 Goal

Develop practical SOC analyst skills and detection engineering experience.
